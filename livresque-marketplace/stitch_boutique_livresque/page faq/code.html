Questions Fréquentes

Pour lire sans connexion internet, vous devez utiliser notre application mobile ou télécharger le livre via le lecteur web pour une utilisation locale temporaire.

  1. Ouvrez l'application Livresque sur votre tablette ou téléphone.
  2. Allez dans Ma Bibliothèque.
  3. Appuyez sur l'icône de téléchargement download à côté du livre.
  4. Une fois le téléchargement terminé, une icône verte apparaîtra.
lightbulb

Note : Les livres téléchargés restent accessibles pendant 30 jours sans reconnexion. Passé ce délai, une brève connexion est nécessaire pour vérifier la licence.
